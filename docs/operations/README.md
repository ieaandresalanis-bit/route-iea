# Daily Operations Guide

How each role should use IEA Growth Intelligence throughout the day.

---

## Director Comercial

### Morning Routine (15 min)

1. **Open AI Brain** (`/sales/ai` → Director tab)
   - Read the executive briefing
   - Check critical alerts — if any, act immediately
   - Review automation health status
   - Note strategic actions for the day

2. **Check War Room** (`/sales/war-room` → Executive tab)
   - Review pipeline value and conversion trends
   - Check for bottlenecks in the funnel
   - Identify underperforming advisors or zones

3. **Review Priorities** (`/sales/ai` → Priorities tab)
   - See top hot leads — ensure they're being worked
   - Check deals to push — confirm advisors are closing

### Weekly Review (30 min)

1. **War Room → Advisors tab:** Compare advisor performance
2. **War Room → Zones tab:** Assess territory coverage
3. **War Room → Strategy tab:** Review and approve strategic recommendations
4. **Campaign Intelligence** (`/sales/intelligence`): Review campaign ROI
5. **Automation Performance** (`/sales/auto-perf`): Check automation health and act on recommendations

### Monthly Review (1 hour)

1. **Strategy** (`/sales/strategy`): Review full commercial plan
2. **Work Plan → Monthly:** Review monthly projections
3. **Campaign Intelligence → Breakdown:** Deep-dive by center, industry, advisor
4. Adjust strategy based on findings

## Supervisor

### Morning Routine (20 min)

1. **Open Daily Ops** (`/sales/ops`)
   - Check daily summary KPIs (follow-ups, overdue, inactive, new)
   - Review advisor priority lists — ensure balance

2. **Check Alert Center** (`/sales/alert-center`)
   - Review open alerts, especially critical and high severity
   - Escalate or reassign as needed
   - Resolve alerts that have been addressed

3. **Check Execution Engine** (`/sales/execution`)
   - Verify task generation ran successfully
   - Review task completion rates from yesterday
   - Reassign tasks for absent advisors

### Throughout the Day

4. **Follow-Up Automation** (`/sales/followup` → Dashboard)
   - Verify sequences are executing
   - Pause sequences for leads that need personal attention
   - Check learning insights for optimization opportunities

5. **Work Plan** (`/sales/plan`)
   - Verify each advisor has a clear daily plan
   - Balance workload if any advisor is overloaded

### When an advisor needs help

6. **Open AI Brain → Coach tab**
   - Enter the lead ID they're struggling with
   - Generate coaching and share with the advisor
   - Use the quick wins as conversation starters

---

## Advisor (Asesor)

### Morning Routine (10 min)

1. **Check Work Plan** (`/sales/plan` → Daily tab)
   - See today's critical tasks, follow-ups, and priority leads
   - Plan your day around the highest-priority items

2. **Check Execution Engine** (`/sales/execution`)
   - View your assigned tasks
   - Start working through them in priority order

### Working Through the Day

3. **For each lead interaction:**
   - If unsure what to say → **Sales Coach** (`/sales/coach`)
     - Select the stage, tone, and channel
     - Use the generated messages
   - After each interaction → mark the task as completed with outcome
   - If pipeline stage changed → update in lead detail

4. **Check Alerts** (`/sales/alerts`)
   - Acknowledge alerts you're aware of
   - Resolve alerts after taking action
   - Escalate alerts you can't handle

### End of Day (5 min)

5. **Review completed tasks** in Execution Engine
6. **Check tomorrow's plan** in Work Plan
7. **Note any leads** that need special attention

## When to Use Each Module

| Situation | Go To |
|-----------|-------|
| "How is the business doing?" | Dashboard (`/sales`) or War Room (`/sales/war-room`) |
| "What should I do today?" | Work Plan (`/sales/plan`) or Execution Engine (`/sales/execution`) |
| "Which leads need attention?" | Daily Ops (`/sales/ops`) or AI Brain → Priorities |
| "What should I say to this lead?" | Sales Coach (`/sales/coach`) or AI Brain → Coach |
| "Are there problems I'm missing?" | Alert Center (`/sales/alert-center`) |
| "How are my campaigns performing?" | Campaign Intelligence (`/sales/intelligence`) |
| "Where are my leads geographically?" | Commercial Map (`/sales/map`) |
| "How is our automation doing?" | Follow-Up (`/sales/followup`) or Auto Perf (`/sales/auto-perf`) |
| "What's the AI's take on things?" | AI Brain (`/sales/ai`) |
| "What should our strategy be?" | Strategy (`/sales/strategy`) or War Room → Strategy |
| "Which cold leads can we recover?" | AI Brain → Reactivation |
| "Where do our leads come from?" | Campaign Attribution (`/sales/attribution`) |
